# Vale Produção - Artist Flow Manager

## Fase 2
Sistema estático em HTML, CSS e JavaScript para publicação direta no GitHub Pages.

### Build
- Versão: v2.0.0
- Data/hora da build: 2026-03-27 11:31:09 -03:00

## O que esta fase entrega
- Dashboard operacional
- Cadastro de artistas e bandas
- Cadastro de singles, EPs, álbuns e campanhas
- Alertas internos por prazo e checklist
- Área de integrações configuráveis
- Exportação JSON e CSV
- Templates de Google Apps Script dentro do ZIP

## Publicação no GitHub Pages
1. Extraia o ZIP.
2. Suba todos os arquivos para um repositório no GitHub.
3. Ative o GitHub Pages apontando para a branch principal e a pasta raiz.
4. O arquivo principal é `index.html`.

## Observação importante
As integrações automáticas com Google Forms, Sheets e Calendar dependem de configuração manual do Apps Script incluso neste pacote.
