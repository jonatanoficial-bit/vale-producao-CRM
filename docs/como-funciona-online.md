# Como o acesso online funciona

## Antes
Cada navegador tinha seus próprios dados.

## Agora
Todos usam o mesmo banco online.
- admin: visão global
- artista: visão individual

## Exemplo
1. o cantor cria conta
2. você entra como admin
3. vincula a conta em Artistas
4. cria o projeto escolhendo esse artista
5. quando ele entra, o projeto já aparece
