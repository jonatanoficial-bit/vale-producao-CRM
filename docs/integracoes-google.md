# Integrações Google - Fase 5

## O que já vem pronto
- Campo para URL do Web App do Apps Script
- Botão para enviar projetos ao Apps Script
- Botões para abrir formulários e calendário
- Geração de link de evento do Google Calendar por projeto

## Fluxo recomendado
1. Criar ou usar seus Google Forms
2. Publicar o Apps Script como Web App
3. Colar a URL do deploy na área Integrações do sistema
4. Salvar os links
5. Usar o botão "Enviar projetos ao Apps Script"

## Sugestões de automação no Apps Script
- Criar eventos no Google Calendar
- Enviar e-mails de alerta
- Gravar dados em Google Sheets
- Gerar gatilhos por data crítica

## Limitação atual
Este projeto permanece estático para funcionar no GitHub Pages sem custo.
Portanto, qualquer automação real depende do Apps Script publicado na sua conta Google.
